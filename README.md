# The Minimalist Pokédex

Premium, Apple-inspired Pokémon search experience built with React, TypeScript, Vite, Tailwind CSS, TanStack React Query, Zustand, and Framer Motion.

## Features
- Search Pokémon by name using the PokéAPI
- Apple-style skeleton loading and elegant inline errors
- Product-style card with official artwork, types, and animated stats
- Shiny toggle without refetching
- Dynamic theme accent based on primary Pokémon type

## Scripts
- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`
